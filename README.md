# Test Codes

JS files can be run using the following commands:

   ```sh
    $ cd /path/to/this/project
    $ node sample_1.js
    $ node sample_2.js
   ```
To check the text file, please run the command below:

   ```sh
    $ cd /path/to/this/project
    $ export FILE_NAME=sample_3.txt
    $ node sample_2.js
   ```
